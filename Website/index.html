Project Overview

Conductive Ink Printer (C.I.P)

Revolutionizing rapid prototyping with conductive ink technology

Project Description

The conductive ink printer (C.I.P) project aims to develop a versatile and efficient system for through additive manufacturing of electronic circuits using conductive ink. This innovative solution will enable designers, engineers, and hobbyists to quickly create and test circuit designs without the need for traditional PCB manufacturing processes. The C.I.P system will feature a user-friendly interface, precision control, and compatibility with various conductive ink formulations, making it an essential tool for accelerating the design and development of electronic devices.

Key Features

  • Rapid prototyping capabilities
  • Conductive ink application system
  • Precision control and automation
  • User-friendly interface

Project Goals

The goals of the C.I.P project are to develop and additve manufacturing method of creating PCBs using condutive ink. We aim to develop multilayer pcb capabilities, using a swapable head system, cutom slicing software, and camera verification of each layer to ensure the board is designed correctly.
